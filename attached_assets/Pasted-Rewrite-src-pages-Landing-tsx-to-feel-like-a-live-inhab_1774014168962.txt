## Data to Fetch on Mount (parallel)

const GATEWAY = import.meta.env.VITE_GATEWAY_URL ?? "";

Fetch these in parallel on mount and refresh every 20 seconds:

const [feedRes, gangsRes, planetsRes] = await Promise.all([
  fetch(`${GATEWAY}/live-feed?limit=6`),
  fetch(`${GATEWAY}/gangs`),
  fetch(`${GATEWAY}/api/planets`),
]);

const { events, stats } = await feedRes.json();
const { gangs } = await gangsRes.json();
const { planets } = await planetsRes.json();

State:
  events     — last 6 live events
  stats      — { total_agents, total_gangs, top_agents }
  gangs      — all gangs sorted by rep
  planets    — all planets with agent_count
  quoteIndex — cycles through recent chat quotes every 4s

## Implementation Notes

- Use React hooks: useState, useEffect, useRef
- Quote cycling: useEffect with setInterval(4000), cleanup on unmount
- War countdown: useEffect with setInterval(1000) on wars array
- Stat numbers: animate count-up on first load (simple CSS or requestAnimationFrame)
- All external links (GitHub) use target="_blank" rel="noreferrer"
- Keep all existing routes — this only replaces the Landing component body
- VITE_GATEWAY_URL must be set in Replit env for fetches to work
